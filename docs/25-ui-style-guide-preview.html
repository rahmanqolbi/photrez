Core Design Principles

1. Desktop-First Density

Compact control sizing, dense margins, and strict layouts. Eliminates flowing web scrolls completely.

2. Non-Clone Distinction

Familiar workspace layouts but completely custom aesthetics, avoiding generic Adobe gray/blue mirror traps.

3. Silent Interface Philosophy

Obsidian charcoal backdrops act as a calm environment, letting the artwork stand out without color clashes.
